# Changelog

All notable changes to this project are documented in this file.

The format follows [Keep a Changelog](https://keepachangelog.com/en/1.1.0/), and
this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2026-07-11

### Added

- Probe-first adaptive HLS transcoding with automatic and explicit ladders.
- Typed source metadata, validation errors, warnings, progress events, dry runs,
  cancellation, reusable clients, and a fluent job builder.
- Audio extraction; alternate audio and WebVTT/SRT subtitle HLS renditions.
- Master and media playlist parsing, serialization, and safe alternate-track
  removal.
- Thumbnail generation, configurable target frame rate, and libx264 presets.
- Runnable framework recipes for Express, Fastify, NestJS, and Next.js.
